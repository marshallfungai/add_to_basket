== Installation ==

1. Register account from https://addtobasket.net
2. Get secret key tied to your domain
3. search for Add to Basket from your wordpress plugins store
4. Alternatively, download plugin from github (https://github.com/marshallfungai/add_to_basket) and upload to the `/wp-content/plugins/` directory
5. Activate the plugin through the 'Plugins' menu in WordPress
6. Enter secret key in the "Add to basket" admin page
